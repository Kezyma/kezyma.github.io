Sierra Madre Revisited

A small mod to allow the player to return to the Sierra Madre casino after completing Fallout New Vegas' Dead Money expansion.

Dead Money is set in the Sierra Madre casino, but once the story is complete, you cannot return to the casino to explore or gamble.

This mod makes three small changes to allow the player to return.

  • Moves a normally hidden door to the back of the Abandoned BOS Bunker, which will return you to the Sierra Madre.
  • Changes the script for exiting the Sierra Madre so teleport the player to the Abandoned BOS Bunker immediately if Dead Money has already been completed.
  • Changes the script that allows ghost people to be killed so that it still runs after Dead Money has been completed.

Installation

To install Sierra Madre Revisited, you first need to download it. The mod is available on NexusMods.

NexusMods

You can install it with a mod manager of your choice, or extract the zip to Fallout New Vegas' Data folder.

Uninstall

To uninstall Sierra Madre Revisited, delete the mod from Fallout New Vegas' Data folder or from your mod manager.
