Reinstaller

for Mod Organizer 2

Introduction

Reinstaller is a plugin for Mod Organizer 2, allowing users to back up downloaded mod installers and run them on demand.

Some of the things you can do with Reinstaller:

  • Back up files from Mod Organizer's downloads folder.
  • Quickly install backed up files from a conveninent menu.
  • Store multiple versions of the same mod and select the preferred file on install.
  • Keep Mod Organizer's downloads folder clean of clutter.
Installation

To install Reinstaller, you will first need to download it, either from the Nexus or from the Github releases page for Reinstaller. Make sure to download the latest version.

Nexus Github

Once you've downloaded the Reinstaller zip file, you will need to extract the reinstaller folder from it and place it in the Mod Organizer plugins folder. If Mod Organizer is installed to C:\Mod Organizer\ then the plugins folder is located at C:\Mod Organizer\plugins\

If you have done this correctly, inside C:\Mod Organizer\plugins\reinstaller\ you should find two folders, shared and reinstaller as well as a file called __init__.py

Reinstaller is now installed and will start the next time you run Mod Organizer.

Alternatively, you can use my Mod Organizer Setup Tool to install a fresh copy of Mod Organizer with Reinstaller already installed.

Usage

Reinstaller adds a selection of tools to Mod Organizer's menu, three quick buttons for adding new installers, running them and deleting them, as well as the main Reinstaller menu.

Create

Presents a dialog to first select a file from Mod Organizer's downloads folder, followed by the option to set a custom name for the download. Once completed, the download will be backed up to Reinstaller.

After the file is added to Reinstaller, it can be safely deleted from Mod Organizer's downloads.

Install

Presents a dialog to select a mod from Reinstaller. If there are multiple files for this mod, a second option will be presented to select the specific file. Once completed, the file will be installed using Mod Organizer's install process.

Delete

Presents a dialog to select a mod from Reinstaller. If there are multiple files for this mod, a second option will be presented to select the specific file. Once completed, the file will be deleted. If this is the last file for a mod, the mod will be removed from Reinstaller.

The Reinstaller menu allows you to browse all existing mods in Reinstaller as well as adding, installing and deleting them.

Settings
Setting Default
enabled true
Determines whether Reinstaller is enabled.
Uninstallation

To remove Reinstaller from Mod Organizer, two folders need to be deleted from Mod Organizer's plugins folder, reinstaller and data\reinstaller. Once deleted, Reinstaller will no longer be installed the next time Mod Organizer starts.

If Mod Organizer is installed to C:\Mod Organizer\ then these folders are located at C:\Mod Organizer\plugins\reinstaller\ and C:\Mod Organizer\plugins\data\reinstaller\
