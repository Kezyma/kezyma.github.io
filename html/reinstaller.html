Reinstaller

Reinstaller is a plugin for Mod Organiser 2 that allows you to backup and reinstall downloaded files without cluttering the downloads panel.

When working with large modlists, it's quite common to need to reinstall a mod to pick different compatibility options. Finding the mod in a relatively cluttered downloads folder, assuming it wasn't deleted accidentally, can be a challenge.

There are also large multi-patch installers that are designed to be run every time you change your mod list, which also have no place to be stored except left in the downloads folder.

Reinstaller helps with this by allowing you to save mods from the downloads folder and re-run the installation for them from the Tools menu.

Installation

To install Reinstaller, you first need to download it. The plugin is available both on NexusMods and Github.

NexusMods Github

Once it is downloaded, browse to where Mod Organizer is installed and open the plugins folder. Extract the reinstaller folder from zip file into plugins to install Reinstaller.

To confirm that Reinstaller has been installed correctly, launch Mod Organizer, click on Tools and then Tool Plugins. Reinstaller should be listed here.

Instructions

Uninstall

To uninstall Reinstaller, delete reinstaller and data\reinstaller from Mod Organizer's plugins folder.
