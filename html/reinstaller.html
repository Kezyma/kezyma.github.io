Reinstaller

Reinstaller is a plugin for Mod Organiser 2 that allows you to backup and reinstall downloaded files without cluttering the downloads panel.

When working with large modlists, it's quite common to need to reinstall a mod to pick different compatibility options. Finding the mod in a relatively cluttered downloads folder, assuming it wasn't deleted accidentally, can be a challenge.

There are also large multi-patch installers that are designed to be run every time you change your mod list, which also have no place to be stored except left in the downloads folder.

Reinstaller helps with this by allowing you to save mods from the downloads folder and re-run the installation for them from the Tools menu.

Installation

To install Reinstaller, you first need to download it. The plugin is available both on NexusMods and Github.

NexusMods Github

Once it is downloaded, browse to where Mod Organizer is installed and open the plugins folder. Extract the reinstaller folder from zip file into plugins to install Reinstaller.

To confirm that Reinstaller has been installed correctly, launch Mod Organizer, click on Tools and then Tool Plugins. Reinstaller should be listed here.

Instructions

Reinstaller adds a selection of items to the tools menu in Mod Organizer.

Opening the main Reinstaller option in the tools menu will open the full Reinstaller manager menu.

Saving Installers

To make a copy of a mod to reinstall later;

  • Select the file from the dropdown list at the top of the Reinstaller manager menu. This lists all files currently in Mod Organizer's downloads folder.
  • A name will be automatically generated, however you can change the name of the installer using the text box to the left of the dropdown list. Giving multiple installers the same name will group them together and give you the option of which to run when installing them later.
  • Once ready, click Add to save the installer, it will then appear in the list below.

Alternatively, click the Create option in the tools menu to get the option to quickly add an installer without openinig the manager menu.

Running Installers

To run an installer that was saved in Reinstaller;

  • Select the installer from the main list in the Reinstaller manager menu.
  • If multiple files have the same name, select the file you wish to install from the dropdown list at the bottom of the Reinstaller manager menu.
  • Click Install to trigger Mod Organizer 2 to install the file.

Alternatively, click the Install option in the tools menu to get the option to quickly select an installer to run without opening the manager menu. If there are multiple files for the selected installer, you will be prompted to select which you wish to install.

Deleting Installers

To delete an installer that was saved in Reinstaller;

  • Select the installer from the main list in the Reinstaller manager menu.
  • If multiple files have the same name, select the file you wish to delete from the dropdown list at the bottom of the Reinstaller manager menu.
  • Click Delete to remove that file.

Alternatively, click the Delete option in the tools menu to get the option to quickly select an installer to remove without opening the manager menu. If there are multiple files for the selected installer, you will be prompted to select which you wish to delete.

Uninstall

To uninstall Reinstaller, delete reinstaller and data\reinstaller from Mod Organizer's plugins folder.
