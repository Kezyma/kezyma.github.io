Profile Sync

for Mod Organizer 2

Introduction

Profile Sync is a plugin for Mod Organizer 2, it allows you to maintain the same mod order (while keeping the enabled/disabled state) across multiple profiles.

Some of the things you can do with Profile Sync:

  • Keep your mod order synchronised across multiple profiles.
  • Create multiple sync groups to have different sets of profiles with different mod orders.
  • Exclude individual profiles that you want to customise and test.
  • Save time arranging the same mod in every profile each time you install a new one.
Installation

To install Profile Sync, you will first need to download it, either from the Nexus or from the Github releases page for Reinstaller. Make sure to download the latest version.

Nexus Github

Once you've downloaded the Profile Sync zip file, you will need to extract the profilesync folder from it and place it in the Mod Organizer plugins folder. If Mod Organizer is installed to C:\Mod Organizer\ then the plugins folder is located at C:\Mod Organizer\plugins\

If you have done this correctly, inside C:\Mod Organizer\plugins\profilesync\ you should find two folders, shared and profilesync as well as a file called __init__.py

Profile Sync is now installed and will start the next time you run Mod Organizer.

Usage

Profile Sync is added to the tools menu in Mod Organizer.

Opening Profile Sync presents you with a dialog for managing sync groups.

Sync Groups

A sync group is a group of profiles that will synchronise their mod order. You create one by giving it a name and clicking Add. Once added, sync groups can be selected in the dropdown list and deleted with the Delete button.

Once you've created a sync group, you can add profiles to the sync group by checking them in the list below. A profile can only be in one sync group at a time. Make sure to add your profiles in the correct order as there is no way to restore a lost mod order;

  • The first time a profile is added to a sync group, a separate mod list will be created for the sync group based on the mod order of the profile being added.
  • Each profile added after this will immediately have its mod list updated to match the group list.
  • When the last profile is removed from a sync group, the group list is deleted. The next profile added will be treated as the first profile added.
Updating

To keep profiles synchronised, Profile Sync does the following;

  • Each time you change the mod order for a profile in a sync group, the group list will be updated.
  • Each time you load (or switch to) a profile, that profile's list will be updated to match the group list.

Profile Sync only updates the order that mods (and separators) are listed in Mod Organizer. This does not change which mods are enabled/disabled or the plugin order.

Settings
Setting Default
enabled true
Determines whether Profile Sync is enabled.
Uninstallation

To remove Profile Sync from Mod Organizer, two folders need to be deleted from Mod Organizer's plugins folder, profilesync and data\profilesync. Once deleted, Reinstaller will no longer be installed the next time Mod Organizer starts.

If Mod Organizer is installed to C:\Mod Organizer\ then these folders are located at C:\Mod Organizer\plugins\profilesync\ and C:\Mod Organizer\plugins\data\profilesync\
