Shortcutter

for Mod Organizer 2
Introduction

Shortcutter is a plugin for Mod Organizer 2, allowing users to create instance and profile specific desktop shortcuts to quickly launch apps through Mod Organizer.

Some of the things you can do with Shortcutter:

  • Create profile specific shortcuts to launch apps through Mod Organizer with a specific profile.
  • Create instance specific shortcuts that will switch to the right instance, so you don't have to (doesn't work for portable instances).
  • Pick custom icons for your shortcuts.
  • Easily run all your different mod configurations conveniently, without having to open Mod Organizer.
Installation

To install Shortcutter, you will first need to download it, either from the Nexus or from the Github releases page for Shortcutter. Make sure to download the latest version.

Nexus Github

Once you've downloaded the Shortcutter zip file, you will need to extract the shortcutter folder from it and place it in the Mod Organizer plugins folder. If Mod Organizer is installed to C:\Mod Organizer\ then the plugins folder is located at C:\Mod Organizer\plugins\

If you have done this correctly, inside C:\Mod Organizer\plugins\shortcutter\ you should find two folders, shared and shortcutter as well as a file called __init__.py

Shortcutter is now installed and will start the next time you run Mod Organizer.

Usage

Shortcutter adds a tool to Mod Organizer's menu to open the Shortcutter window.

The Shortcutter window allows you to select a profile and an application from the current Mod Organizer instance, then select an icon from an ico or exe file and finally name your shortcut. Clicking Ok will create that shortcut on your desktop.

Settings
Setting Default
enabled true
Determines whether Shortcutter is enabled.
Uninstallation

To remove Shortcutter from Mod Organizer, a folder needs to be deleted from Mod Organizer's plugins folder, shortcutter. Once deleted, Shortcutter will no longer be installed the next time Mod Organizer starts.

If Mod Organizer is installed to C:\Mod Organizer\ then this folder is located at C:\Mod Organizer\plugins\shortcutter\
