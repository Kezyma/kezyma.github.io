Shortcutter

Shortcutter is a plugin for Mod Organiser 2 that allows you to create instance and profile specific shortcuts to launch different mod lists without openinig Mod Organizer.

It can often be time consuming to launch Mod Organizer, switch to the instance and then profile for the game you want to play before you can launch it.

Shortcutter allows you to create a direct shortcut to launch a game using a specific instance and profile without having to ever open Mod Organizer yourself.

Installation

To install Shortcutter, you first need to download it. The plugin is available both on NexusMods and Github.

NexusMods Github

Once it is downloaded, browse to where Mod Organizer is installed and open the plugins folder. Extract the shortcutter folder from zip file into plugins to install Shortcutter.

To confirm that Shortcutter has been installed correctly, launch Mod Organizer, click on Tools and then Tool Plugins. Shortcutter should be listed here.

Instructions

Shortcutter adds an item to the tools menu in Mod Organizer.

Opening the main Shortcutter option in the tools menu will open the Shortcutter menu.

To create an instance and profile specific shortcut;

  • Select the Mod Organizer profile from the Profiles dropdown list.
  • Select which application should be launched from Mod Organizer's applications list.
  • Optionally select an icon for the shortcut.
  • A name will be automatically generated for the shortcut, but it can be changed using the Name text box.
  • Click Create and a shortcut to launch the selected application using the current profile will be added to your desktop.

If you have multiple instances, Shortcutter will detect this and always switch to the correct one before launching.

Uninstall

To uninstall Shortcutter, delete shortcutter from Mod Organizer's plugins folder.
