List Exporter

for Mod Organizer 2

Introduction

List Exporter is a plugin for Mod Organizer 2 that exports your mod list to various formats for sharing or documentation purposes.

Some of the things you can do with List Exporter:

  • Export your mod list to HTML, Markdown, CSV, or JSON formats.
  • Include mods from multiple profiles in a single export.
  • Organize exports by category using your existing separators.
  • Choose which columns to include and customize their order.
  • Automatically generate NexusMods links for mods with valid mod IDs.
  • Exclude specific mods from exports using a simple tag.
Installation

To install List Exporter, you will first need to download it from the Github releases page. Make sure to download the latest version.

Github

Once you've downloaded the List Exporter zip file, extract the listexporter folder and place it in the Mod Organizer plugins folder. If Mod Organizer is installed to C:\Mod Organizer\ then the plugins folder is located at C:\Mod Organizer\plugins\

If you have done this correctly, inside C:\Mod Organizer\plugins\listexporter\ you should find two folders, shared and listexporter as well as a file called __init__.py

List Exporter is now installed and will start the next time you run Mod Organizer.

Alternatively, if you have Plugin Finder installed, you can install List Exporter directly from within Mod Organizer.

Usage

List Exporter can be opened through a new link added to Mod Organizer's tools menu.

Export Format

Select the format for your export using the radio buttons at the top. Each format has different use cases:

  • HTML - A styled web page suitable for sharing online or viewing in a browser.
  • Markdown - Plain text tables ideal for GitHub, Discord, forums, or README files.
  • CSV - Spreadsheet format compatible with Excel, Google Sheets, or other applications.
  • JSON - Structured data format for programmatic use or data processing.
Options

Enable Separate table for each category to split your export into multiple tables, one for each category defined by your separator mods. When disabled, all mods appear in a single table with a Category column.

Profile Selection

Select one or more profiles to include in the export. A mod will be included if it is enabled in any of the selected profiles. When multiple profiles are selected, additional columns show the enabled/disabled status for each profile.

Use the Select All and Select None buttons to quickly manage profile selection.

Column Selection

Choose which columns to include in your export by checking or unchecking them in the column list. You can also drag columns to reorder them - the export will use the order shown in the list.

Available columns:

  • Mod Name - The name of the mod, with a hyperlink to NexusMods if a mod ID is available.
  • Category - The category the mod belongs to (only shown when not using separate tables).
  • Notes - Notes from the mod's meta.ini file.
  • Comments - Comments from the mod's meta.ini file.
  • Profile columns - When multiple profiles are selected, shows enabled status per profile.
Excluding Mods

To exclude a specific mod from all exports, add <LE_Exclude> anywhere in the mod's Notes or Comments field in Mod Organizer. The tag is not case-sensitive.

This is useful for hiding utility mods, merged patches, or other mods you don't want to appear in shared mod lists.

Exporting

Click the Export button to generate your mod list. You will be prompted to choose a save location and filename. The appropriate file extension will be added based on your selected format.

Export Formats
HTML Export

Creates a complete HTML page with Bootstrap styling. The page includes a responsive table with your mod list, styled headers for categories (if using separate tables), and clickable links to NexusMods where available. This format is ideal for sharing on websites or viewing locally in a browser.

Markdown Export

Generates Markdown-formatted tables that render nicely on GitHub, GitLab, Discord, Reddit, and many forums. Category headers use Markdown heading syntax. This format is perfect for README files, wiki pages, or sharing in communities that support Markdown.

CSV Export

Produces a comma-separated values file that can be opened in Excel, Google Sheets, LibreOffice Calc, or any spreadsheet application. When using separate categories, each category is preceded by a header row. This format is useful for further data analysis or manipulation.

JSON Export

Outputs structured JSON data containing all mod information. When using separate categories, the JSON is organized by category. This format is ideal for developers who want to process the mod list programmatically or integrate it with other tools.

Settings
Setting Default
enabled true
Determines whether List Exporter is enabled.
defaultformat html
The default export format when opening the List Exporter dialog. Valid values are: html, markdown, csv, json.
separatecategories false
Whether to separate tables by category by default when opening the List Exporter dialog.
Uninstallation

To remove List Exporter from Mod Organizer, two folders need to be deleted from Mod Organizer's plugins folder: listexporter and data\listexporter. Once deleted, List Exporter will no longer be installed the next time Mod Organizer starts.

If Mod Organizer is installed to C:\Mod Organizer\ then these folders are located at C:\Mod Organizer\plugins\listexporter\ and C:\Mod Organizer\plugins\data\listexporter\

Alternatively, if you installed List Exporter through Plugin Finder, you can uninstall it from there.
