Myar Aranath English Translation Race Fix

A patch to re-enable the intended playable races when playing SureAI's total conversion mod, Myar Aranath.

Myar Aranath is a total conversion mod for Morrowind, developed by SureAI. The english translation patch for Myar Aranath causes the game to offer Morrowind's default races instead of Myar Aranath's.

This patch re-applies the default Myar Aranath races (and translates them) so that they become playable again.

Installation

To install Myar Aranath English Translation Race Fix, you first need to download it. The mod is available on NexusMods.

NexusMods

You can install it with a mod manager of your choice, or extract the zip to Myar Aranath's Data Files folder.

Your load order should be the same as below, no other esp or esm files should be enabled.

ma_script_base.esm
MA_RoK_Eng v0001.esm
Additional_Journal_Translations_ENG.esp
Myar_Aranath_English_Race_Fix.esp

If you use Morrowind Code Patch, you must also disable Script expression parser fix from the Bug Fixes section, otherwise a script error will break the game.

Uninstall

To uninstall Myar Aranath English Translation Race Fix, delete the mod from Myar Aranath's Data Files folder or from your mod manager.
