UI Remastered

UI Remastered is a complete texture replacement for every aspect of Morrowind's user interface.

This mod replaces textures for many aspects of Morrowind's UI, with support for a variety of other mods. Everything has been designed to keep within the original UI colour scheme and all the packages are modular so you can install whichever parts of UI Remastered that you want.

There is also an optional silver edition of UI Remastered in which all UI and HUD elements are silver instead of gold.

Installation

To install UI Remastered, you first need to download it. The mod is available on NexusMods.

NexusMods

Mod Manager Install

It is best to install UI Remastered through a mod manager, as it is packaged as a FOMOD installer.

Edition

The edition section allows you to select between the original and silver versions of UI Remastered.

  • Original Edition is UI Remastered in traditional Morrowind colours, with gold borders and elements.
  • Silver Edition is UI Remastered, redone with silver borders and elements. If you select the Silver Edition

Packages

The packages section allows you to select which UI elements you would like to install, you can select any or none of them.

  • Main Menu Buttons replaces the buttons on both the main menu and the escape menu during the game. Supports Continue and MWSE.
  • Book + Scroll Buttons replaces the buttons on the book and scroll with icon buttons inspired by the ones from UI Revamped.
  • Book + Scroll Textures replaces the book and scroll images with new ones.
  • UI & HUD Elements replaces all the borders, buttons and headings in the UI. Supports MGE XE - HUD Mods time display mod.
  • Stat Bars replaces the bars used to display stats, encumberance and health, magicka and stamina on the HUD.
  • Item Icon Frames replaces the icon background for enchanted items as well as providing other new borders for item icons.
  • Splash Screens adds a new set of splash screens based on edited screenshots from the game.

Cursors

The cursors section allows you to install replacement cursors, as well as an alternative main cursor.

  • Cursors Pack replaces all the cursors in the game.
  • Alternative Main Cursor replaces the main cursor in the Cursors Pack with a slimmer one.

Crosshairs

There are two optional crosshairs that can be installed, a vanilla style one or a more modern one. Both crosshairs come with an ownership indicator version, for these to work you will either need to install Ownership Indicator or MGE XE - HUD Mods ownership indicator.

  • Morrowind Crosshair is a smaller version of the vanilla crosshair.
  • Skywind Crosshair is a modern style alternative crosshair created by Left4No.

Skill + Magic Icons

This section contains replacers for all of the skill and magic icons in the game, there are two main flavours to choose from. All options contain support for Magicka Expanded as well as UI Expansion's inventory filters.

  • Borderless Icons are cutouts of the symbols from the vanilla icons, coloured in the vanilla style.
  • Borderless Transparent Icons are the Borderless Icons, but with a transparent background.
  • Smooth Icons are smoothened versions of the vanilla icons, with a more painted or flat style.

Item Icons

This section contains replacers for all of the item icons in the game, based on Ultimate Icon Replacer, there are two main flavours to choose from. Both packs contain replacements for many of the icons in Tamriel Data.

  • Crisp Item Icons are sharper, more detailed and higher contrast versions of the icons.
  • Material Item Icons are flatter version of the icons that resemble material UI design.

Compatibility

This section allows you to install a compatibility patch to work with OpenMW. Do not select this option if you are using the original game engine or it will break the UI.

Manual Install

It is not suggested to install UI Remastered manually, however it can still be done.

Open the UI Remastered zip file, each folder within it corresponds to one of the options in the FOMOD installer. Open the folders for each option you wish to install and copy the contents to Morrowind's Data Files folder.

Each folder starts with a number, if two folders share the same number, they are alternative options and only one should be picked.

Recommended Mods

The following are a list of mods that UI Remastered was designed to work with.

  • Morrowind Code Patch contains a setting required to use the ownership indicator mod.
  • Ownership Indicator enables the crosshair to change when looking at owned items.
  • UI Expansion features a complete overhaul of the UI with many quality of life improvements.
  • Quickloot adds a popup loot menu when looking at containers.
  • Hotkeys Extended adds many more hotkey options.
  • HUD Weapon Charge displays the currently equipped weapon's charge on the HUD.
  • Animated Main Menu for Morrowind is the best looking main menu replacer.
  • Better Dialogue Font replaces the regular text font with a much sharper one.
  • Better Daedric Font replaces the daedric text font with a much sharper one.
