OpenMW Player

for Mod Organizer 2

Introduction

OpenMW Player is a plugin for Mod Organizer 2 that automatically exports your mod list, enabled plugins, and grass mods to OpenMW whenever you run the game through Mod Organizer.

Some of the things you can do with OpenMW Player:

  • Run OpenMW through Mod Organizer with your current mod list and plugins.
  • Specify Groundcover mods to use with OpenMW with different setups for each profile.
  • Select the OpenMW.cfg location that you want to update with your mod list.
Installation

To install OpenMW Player, you will first need to download it, either from the Nexus or from the Github releases page for OpenMW Player. Make sure to download the latest version.

Nexus Github

Once you've downloaded the OpenMW Player zip file, you will need to extract the openmwplayer folder from it and place it in the Mod Organizer plugins folder. If Mod Organizer is installed to C:\Mod Organizer\ then the plugins folder is located at C:\Mod Organizer\plugins\

If you have done this correctly, inside C:\Mod Organizer\plugins\openmwplayer\ you should find two folders, shared and openmwplayer as well as a file called __init__.py

OpenMW Player is now installed and will start the next time you run Mod Organizer.

You should now add OpenMW.exe and/or OpenMW-Launcher.exe to the list of executables in Mod Organizer 2 and run OpenMW from there. Whenever OpenMW.exe, OpenMw-CS.exe or OpenMW-Launcher.exe are run through Mod Organizer, your mod list and enabled plugins will be automatically applied to OpenMW.

Usage

OpenMW Player adds a tool to Mod Organizer's menu to open the Shortcutter window.

The OpenMW Player window allows you to select the location of your openmw.cfg file, as well as select all of your grass/groundcover mods. Any groundcover/grass mods should be selected here as the plugins should not be enabled when the game runs. The selected groundcover mods are set per-profile, so you can have a different set of groundcover mods for each profile in Mod Organizer.

Please note, you will still need to add lines to OpenMW's settings.cfg to enable groundcover. The following are an example of what you may need to add, taken from Remiros' Groundcover: 

                        
[Groundcover]
enabled = true
density = 1.0
min chunk size = 0.5
stomp mode = 2
stomp intensity = 2
Settings
Setting Default
enabled true
Determines whether OpenMW Player is enabled.
openmwcfgpath Default OpenMW.cfg Path
The path to openmw.cfg.
Uninstallation

To remove OpenMW Player from Mod Organizer, a folder needs to be deleted from Mod Organizer's plugins folder, openmwplayer. Once deleted, OpenMW Player will no longer be installed the next time Mod Organizer starts.

If Mod Organizer is installed to C:\Mod Organizer\ then this folder is located at C:\Mod Organizer\plugins\openmwplayer\
