Curation Club

for Mod Organizer 2

Introduction

Curation Club is a plugin for Mod Organizer 2, allowing users to import Creation Club content into Mod Organizer as separate mods.

Some of the things you can do with Curation Club:

  • Organise Creation Club content directly within Mod Organizer.
  • Import all Creation Club content into Mod Organizer.
  • Customise the name for Creation Club content.
  • Move Creation Club manifest files using Root Builder support.
Installation

To install Curation Club, you will first need to download it, either from the Nexus or from the Github releases page for Curation Club. Make sure to download the latest version.

Nexus Github

Once you've downloaded the Curation Club zip file, you will need to extract the curationclub folder from it and place it in the Mod Organizer plugins folder. If Mod Organizer is installed to C:\Mod Organizer\ then the plugins folder is located at C:\Mod Organizer\plugins\

If you have done this correctly, inside C:\Mod Organizer\plugins\curationclub\ you should find two folders, shared and curationclub as well as a file called __init__.py

Curation Club is now installed and will start the next time you run Mod Organizer.

Usage

Curation Club is added to the tools menu in Mod Organizer.

Opening Curation Club presents you with a dialog to modify settings and then move Creation Club content into Mod Organizer.

When run, Curation Club will search the game folder and any enabled mods in Mod Organizer. Any Creation Club content found will be added to a new mod in Mod Organizer. Each mod will be named based on the format provided.

Settings
Setting Default
enabled true
Determines whether Curation Club is enabled.
modnameformat Creation Club - {creation}
Specifies the format for Creation Club mod names. {creation} is the name of the creation itself.
rootbuildersupport false
Enables support for Root Builder. Creation Club manifest files will also be moved to Mod Organizer inside Root folders with their respective mods. Enable if you use Root Builder, disable otherwise.
Uninstallation

To remove Curation Club from Mod Organizer, two folders need to be deleted from Mod Organizer's plugins folder, curationclub and data\curationclub. Once deleted, Reinstaller will no longer be installed the next time Mod Organizer starts.

If Mod Organizer is installed to C:\Mod Organizer\ then these folders are located at C:\Mod Organizer\plugins\curationclub\ and C:\Mod Organizer\plugins\data\curationclub\
