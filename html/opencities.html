Open Cities Classic Flag Remover

A patch to remove flags added to cities by the Open Cities Classic mod for Oblivion.

Open Cities Classic allows players to visit the cities of Oblivion without loading screens, but it also adds flags to every city. While some players might like this, for a more purist modlist, this plugin will remove the flags again.

Installation

To install Open Cities Classic Flag Remover, you first need to download it. The mod is available on NexusMods.

NexusMods

You can install it with a mod manager of your choice, or extract the zip to Oblivion's Data folder.

This mod should be placed after Open Cities Classic in your load order.

Uninstall

To uninstall Open Cities Classic Flag Remover, delete the mod from Oblivion's Data folder or from your mod manager.
