Voices of Vvardenfell

Voices of Vvardenfell adds fully voiced dialogue to Morrowind, generated using ElevenAI.

Morrowind contained some small dialogue lines that are heard as the player walks around, however it lacks voiced dialogue for conversations, leading to a lot of reading being required. Not only is this less immersive than having voiced dialogue, but also makes the game hard or impossible to play for those who struggle with reading.

Voices of Vvardenfell is a project that aims to voice every dialogue line in Morrowind as well as Tribunal, Bloodmoon and the Official Plugins. It also contains a framework that allows modders to add voices to dialogue with no scripting or modding experience required.

Installation

To install Voices of Vvardenfell, you first need to download it. The mod is available on NexusMods.

NexusMods

Requirements

Voices of Vvardenfell requires the following mods to function, ensure they are all installed before proceeding.

  • MGE XE
  • MWSE

Mod Manager Install

It is best to install Voices of Vvardenfell through a mod manager, as it is packaged as a FOMOD installer.

The Voices of Vvardenfell option in the Core section contains the script and all voice files for Morrowind.

The Patch for Purists option in the Patches section contains extra voice files to support Patch for Purists.

The options in the Extras section contain voice files with alternative voices for Almalexia and Neloth, created by FurbyZones.

Manual Install

It is not suggested to install Voices of Vvardenfell manually, however it can still be done.

Open the Voices of Vvardenfell zip file, each folder within it corresponds to one of the options in the FOMOD installer. Open the folders for each option you wish to install and copy the contents to Morrowind's Data Files folder.

Greetings Mode

Voices of Vvardenfell has an optional greetings only mode. This can be enabled from the mod configuration menu. If enabled, only the initial line of dialogue when speaking to an NPC will be played.

Adding Voices

File Structure

Lines are searched for in a priority order, so each voice can have a generic line, with specific NPCs, factions and faction ranks having further lines that can override them. Placing a file in the appropriate location for dialogue in the game will result in it being played whenever that dialogue is triggered.

Voices of Vvardenfell's plugin searches for voices in Morrowind's Data Files folder. The first file found is the one that is played, searching in the following locations.

  • Sound\Vo\AIV\{race}\{sex}\{actor_id}\{faction_id}\{faction_rank}\{info_id}.mp3
  • Sound\Vo\AIV\{race}\{sex}\{actor_id}\{faction_id}\{info_id}.mp3
  • Sound\Vo\AIV\{race}\{sex}\{actor_id}\{info_id}.mp3
  • Sound\Vo\AIV\{race}\{sex}\{faction_id}\{faction_rank}\{info_id}.mp3
  • Sound\Vo\AIV\{race}\{sex}\{faction_id}\{info_id}.mp3
  • Sound\Vo\AIV\{race}\{sex}\{info_id}.mp3
  • Sound\Vo\AIV\creature\{actor_id}\{faction_id}\{faction_rank}\{info_id}.mp3
  • Sound\Vo\AIV\creature\{actor_id}\{faction_id}\{info_id}.mp3
  • Sound\Vo\AIV\creature\{actor_id}\{info_id}.mp3
  • Sound\Vo\AIV\creature\{faction_id}\{faction_rank}\{info_id}.mp3
  • Sound\Vo\AIV\creature\{faction_id}\{info_id}.mp3
  • Sound\Vo\AIV\creature\{info_id}.mp3

Below are some examples for where a voice file for a line with an info_id of 123456.

  • Sound\Vo\AIV\wood elf\f\123456.mp3
    • A generic voice line played by any female Wood Elf.
  • Sound\Vo\AIV\imperial\m\caius cosades\123456.mp3
    • A voice line played by Caius Cosades.
  • Sound\Vo\AIV\dark elf\m\aryon\Telvanni\1\123456.mp3
    • A voice line played by Aryon, exclusively for members of Great House Telvanni at the rank of Retainer.
  • Sound\Vo\AIV\creature\dagoth_ur_1\123456.mp3
    • A voice line played by Dagoth Ur.

Extracting Lines

A tool is available to extract dialogue (including every required part of the file path) from esp and esm files and save it as a csv for easy browsing. The latest version is available on Github.

The Line Extraction tool requires .NET 7.0 to be installed or it will not run. An active internet connection is also required to download tes3conv.

Github

To use the tool, create a folder called Esp next to Kezyma.MorrowindLineExtractor.exe and place the esp and/or esm files you want to extract inside it. You must also include any dependencies of these mods or the extraction will fail.

Run Kezyma.MorrowindLineExtractor.exe and a Csv folder will be created with a csv file for each esp or esm that was extracted. Please note, the InfoId of each row has _ added to the start to prevent Excel from formatting the column incorrectly, this should not be included in the file's name.

Uninstall

To uninstall Voices of Vvardenfell, delete it from your mod manager, or in the case of a manual install, remove the following files/folders from Morrowind's Data Files folder.

  • Textures\AI Voices\
  • Sound\Vo\AIV\
  • MWSE\mods\AI Voices\
