OpenMW Player

for Mod Organizer 2

Introduction

OpenMW Player is a plugin for Mod Organizer 2 that automatically exports your mod list, enabled plugins, and grass mods to OpenMW whenever you run the game through Mod Organizer.

Some of the things you can do with OpenMW Player:

  • Run OpenMW through Mod Organizer with your current mod list and plugins.
  • Specify Groundcover mods to use with OpenMW with different setups for each profile.
  • Select the OpenMW.cfg location that you want to update with your mod list.
  • Manage omwaddon and omwscripts files using dummy esp files.
Installation

To install OpenMW Player, you will first need to download it, either from the Nexus or from the Github releases page for OpenMW Player. Make sure to download the latest version.

Nexus Github

Once you've downloaded the OpenMW Player zip file, you will need to extract the openmwplayer folder from it and place it in the Mod Organizer plugins folder. If Mod Organizer is installed to C:\Mod Organizer\ then the plugins folder is located at C:\Mod Organizer\plugins\

If you have done this correctly, inside C:\Mod Organizer\plugins\openmwplayer\ you should find two folders, shared and openmwplayer as well as a file called __init__.py

OpenMW Player is now installed and will start the next time you run Mod Organizer.

Alternatively, you can use my Mod Organizer Setup Tool to install a fresh copy of Mod Organizer with OpenMW Player already installed.

You should now add OpenMW.exe and/or OpenMW-Launcher.exe to the list of executables in Mod Organizer 2 and run OpenMW from there. Whenever OpenMW.exe, OpenMw-CS.exe or OpenMW-Launcher.exe are run through Mod Organizer, your mod list and enabled plugins will be automatically applied to OpenMW.

Usage

OpenMW Player takes a copy of your openmw.cfg and settings.cfg for your current OpenMW setup and stores them in the current profile in Mod Organizer 2. Each profile has a separate configuration.

  • When openmw.exe is launched through Mod Organizer 2, it uses the config files located in the current profile. If profile-specific save games are enabled, OpenMW will save to the current profile, openmw.exe must be launched directly from Mod Organizer 2 and not through the OpenMW Launcher for this to function.
  • When any other OpenMW executable is launched, the existing openmw.cfg and settings.cfg will be backed up and replaced with the ones from the profile. When the executable closes, they will be moved back and the original files are restored.

OpenMW Player adds a tool to Mod Organizer's menu to open the OpenMW Player window.

The OpenMW Player window allows you to select the location of your openmw.cfg file and import it to the current profile.

There is an additional option to toggle on the use of dummy esp files for .omwaddon and .omwscripts files. If enabled, matching esp files will be created and can be enabled/disabled and sorted within Mod Organizer. Then, when the game is run, the appropriate .omwaddon and .omwscripts files will be enabled in OpenMW instead of the dummy esp files. If enabled, dummy esp files will be created on the following events:

  • When the setting is enabled, dummy files will be created for all mods.
  • When Mod Organizer is started, dummy files will be created for all mods.
  • When a mod is enabled or disabled, dummy files will be created for all mods.
  • When a mod is installed, dummy files will be created for that mod.

The Groundcover tab allows you to select which esp files should be used for groundcover. These should not be enabled in the normal plugins panel and instead be selected here. You will still need to enable groundcover in settings.cfg for this to work.

The Archives tab allows you to select which Bsa archives should be used as well as in what order they should be loaded.

The Settings tab allows you to manage all of the other settings from openmw.cfg.

The Engine tab allows you to manage all of the settings from settings.cfg, these are all of the settings handled by the OpenMW Launcher.

There are also two tools added to the menu.

  • Export will overwrite the current openmw.cfg and settings.cfg with the ones from this profile. This allows you to launch OpenMW from outside of Mod Organizer with the current setup.
  • Import will overwrite this profile's openmw.cfg and settings.cfg with the one selected in the OpenMW Player menu.

Uninstallation

To remove OpenMW Player from Mod Organizer, a folder needs to be deleted from Mod Organizer's plugins folder, openmwplayer. Once deleted, OpenMW Player will no longer be installed the next time Mod Organizer starts.

If Mod Organizer is installed to C:\Mod Organizer\ then this folder is located at C:\Mod Organizer\plugins\openmwplayer\
